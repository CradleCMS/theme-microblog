# Microblog template for Cradle CMS

Designed for web articles written in markdown

## CSS
Minimal CSS approach where Ok.css framework used with minor additions in a separate file.

## Templates

* Index page
* Sub page
* Contact page
* Blog
* Article template with support for: 
  * Sidenotes
  * Footnotes
  * Table of contents
* Blogs list
* 404 page

## Features
* Open graph tags
